Задача 1 (input_user_data.py)

Да се напише програма (input_user_data.py), която да позволява на администратор да въведе следните данни за потребители:

		|--------|-------|----------|
		| name   |   age | city     |
		|--------|-------|----------|
		| Pier   |    40 | Paris    |
		| John   |    25 | New York |
		| Sara   |    45 | London   |
		| Mary   |    30 | Paris    |
		| Bob    |    35 | London   |
		| Mike   |    50 | London   |
		|--------|-------|----------|

Структурата от данни, в която ще съхранявате въведената информация е по ваше усмотрение.

Преди да се въведат данните за всеки потребител, се извежда съобщение "Enter new user data [y/n]?: ". Ако отговора е 'y' или 'Y', се въвежда информация за нов потребител. Ако отговора е различен от 'y' или 'Y', се прекратява въвеждането на данни за потребителите и въведените данни се записват във файл.


python input_user_data.py

	Enter new user data [y/n]?: y
	Enter user name:Pier
	Enter user age: 40
	Enter user city: Paris

	Enter new user data [y/n]?: y
	Enter user name:John
	Enter user age: 25
	Enter user city: New York
	...

	Enter new user data [y/n]?: y
	Enter user name:Mike
	Enter user age: 50
	Enter user city: London

	Enter new user data [y/n]?: n

Формата на файла, в който ще се запишат въведените данни, е по ваше усмотрение (csv, json, ...), така че друга програма да може да чете въведените данни и да ги обработва.
