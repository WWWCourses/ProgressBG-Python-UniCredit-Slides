Data Serialization

Data Serialization

Overview

Data serialization is the concept of converting structured data (lists, dictionaries, objects, etc) into a format (byte stream, XML, JSON,... ) that can be stored (for example, in a file) or transmitted and reconstructed later.

Pickle

The native data serialization module for Python is called Pickle.
It supports "Pickling" - when a Python object hierarchy is converted into a byte stream, and "Unpickling" - the inverse operation, when a byte stream is converted back into an object hierarchy
pickle — Python object serialization

Example


      import pickle

      # let's serialize a simple dict
      prices = { 'apples': 2.50, 'oranges': 1.90, 'bananas': 2.40 }

      #convert the object to a serialized string
      serialized_prices = pickle.dumps( prices )
      print(serialized_prices)

      #de-serialize (unpickle) an object
      received_prices = pickle.loads( serialized_prices )
      print(received_prices)

Notes

For storing data in databases and archival storage, you’re probably better off using a more standard data encoding, such as JSON, XML, CSV

Parse JSON string to Python objects.

json.loads()
Deserialize a string (or bytes or bytearray instance) containing a JSON document into a Python object using next conversion table.
the s in loads() stands for String, not plural.

json.loads() example


      import json
      from operator import itemgetter

      json_str = """
        [
            {
                "name": "apple",
                "price": 1.80
            },
            {
                "name": "orange",
                "price": 2.10
            },
            {
                "name": "bananas",
                "price": 1.60
            }
        ]
      """


      #read json from string
      json_data = json.loads(json_str)

      # print the list sorted by "price"
      for i in sorted(json_data,key=itemgetter("price")):
        print(i)

Parsing CSV

Parsing CSV

Overview

CSVCommaSeparatedValues
the most common import and export format for spreadsheets and databases.
For CSV parsing Python provides the built-in csv module
csv — CSV File Reading and Writing

csv.reader()

csv.reader()
Return a reader object which will iterate over lines in the given csvfile
Algorithm:
Open the file
Create a CSV reader
For every line (row) in the file, do something
If csvfile is a file object, it should be opened with newline=''

Dialects and Formatting Parameters

All csv readers/writers objects can be given a Dialect and/or Formatting Parameters to tune the CSV parsing

Parsing CSV data - example 1 - code


      import csv

      # Open the file
      with open('sample_data.csv',newline='') as f:
        # Create a CSV reader:
        f_csv = csv.reader(f)

        # skip header row:
        headers = next(f_csv)

        # print CSV data, sorted by "Price"
        for row in sorted(f_csv, key=lambda a:a[1]):
          print(row)

        #OUTPUT
        #['AA', '39.48', '6/11/2007', '9:36am', '-0.18', '181800']
        #['C', '53.08', '6/11/2007', '9:36am', '-0.25', '360900']

Print only albums, produced by "Fiction Records" during the period 1980 - 1989 (both included)


      import csv
      with open('TheCure_Discography.csv',newline='') as f:
          f_csv = csv.reader(f, delimiter=';')
          headers = next(f_csv)

          for row in filter(
              lambda l: "Fiction Records" in l[1] and 1980 <= int(l[2]) <= 1989,
              f_csv
          ):
              print(row)

csv.DictReader()

csv.DictReader()
operates like a regular reader but maps the information in each row to an OrderedDict collection.
By default, the header row will be used to set keys and other rows - will be set as a values.

csv.DictReader() - example


      import csv
      with open('sample_data.csv',newline='') as f:
        reader = csv.DictReader(f, delimiter=',')

        for r in reader:
          print(r)

Parsing XML data - sample data


      <data>
          <items>
              <item atr1="value1">item1 data</item>
              <item atr2="value2">item2 data</item>
          </items>
      </data>

Parsing XML data - example


      import xml.etree.ElementTree as ET

      tree = ET.parse('sample_data.xml')
      root = tree.getroot()

      # print items attributes
      print('\nAll attributes:')
      for elem in root:
          for subelem in elem:
              print(subelem.attrib)

      # print items data
      print('\nAll item data:')
      for elem in root:
          for subelem in elem:
              print(subelem.text)

Task2 - A possible, pure Python, Solution


      def rearange_books_by_category(books_json):
        """re-arange books by category.
          returns:
          books =
          {
            "Beginner":[
              {
                "title":
                "author":
                "url"
              }
            ],
            "Intermediate":[
              {
                "title":
                "author":
                "url"
              }
            ],
          }
        """

        rearanged_books = {}
        for book in books_json['books']:
          category = book['level']
          book_data = {
            "title": book['title'],
            "author":book['author'],
            "url":book['url']
          }

          if category in rearanged_books.keys():
            rearanged_books[category].append(book_data)
          else:
            rearanged_books[category] = [book_data]

        return rearanged_books
